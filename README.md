#  GUI-Youtube-Downloader
This application can download audio or video from youtube


## Features
Download multi links at one time \
Give feedback for every download 


## Libraries
Tkinter \
pytube \
os \
PIL 


## Commands to download Libraries
pip install pytube \
pip install tkintertable \
pip install Pillow
